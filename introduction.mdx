## Overview

Wesley N provides a centralized platform where you create, organize, and manage all your project documentation. You build custom documentation spaces tailored to your needs, with features for collaboration, version control, and seamless search. Use it to document APIs, user guides, or internal wikis efficiently.

The platform integrates your brand seamlessly, starting with the primary color `#3B82F6`. You access everything from a clean dashboard at `https://dashboard.wesleyn.com`.

## Brand Guidelines

Maintain consistency with Wesley N's brand using the primary color `#3B82F6`.

<Tabs>
  <Tab title="CSS Usage" icon="code">
    Apply the brand color in your custom themes or embeds.

    <CodeGroup tabs="CSS,SCSS">
    ````css
    :root {
      --primary-color: #3B82F6;
    }

    .button-primary {
      background-color: var(--primary-color);
      color: white;
    }
    ````
    ````scss
    $primary: #3B82F6;

    .button-primary {
      background-color: $primary;
      color: white;
    }
    ````
    </CodeGroup>
  </Tab>
  <Tab title="Integration Tips" icon="settings">
    Use the color for buttons, links, and accents. Ensure WCAG compliance by pairing with sufficient contrast ratios.

    <ul>
      <li>Primary buttons: `#3B82F6` on white</li>
      <li>Hovers: `#2563EB` (darker shade)</li>
      <li>Avoid pure black text on brand color</li>
    </ul>
  </Tab>
</Tabs>

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://wesleyn.com` and create your free account.
  </Step>
  <Step title="Create Space" icon="folder-plus">
    From the dashboard, click "New Space" and name it for your project.
  </Step>
  <Step title="Add Pages" icon="file-plus">
    Start with an `introduction.mdx` file using the provided template.
  </Step>
  <Step title="Customize Theme" icon="palette">
    Edit space settings to apply `#3B82F6` as your primary color.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
````javascript
// Example: Embed Wesley N docs in your app
const embedUrl = 'https://docs.wesleyn.com/your-space/introduction';
fetch(embedUrl)
  .then(response => response.text())
  .then(html => document.getElementById('doc-embed').innerHTML = html);
````
````python
# Python integration example
import requests
response = requests.get('https://docs.wesleyn.com/your-space/introduction')
print(response.text)
````
