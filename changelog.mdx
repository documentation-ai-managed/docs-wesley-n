## Upgrade Process

Follow these steps to upgrade your Wesley N instance safely.

<Callout kind="info">
Backup your documentation space before upgrading. Use the export feature to create a full snapshot.
</Callout>

<Steps>
  <Step title="Backup" icon="database">
    Export your space via the dashboard.

    ```
    curl -X POST https://api.example.com/export \
      -H "Authorization: Bearer YOUR_TOKEN" \
      -d '{"spaceId": "your-space-id"}'
    ```
  </Step>
  <Step title="Update" icon="download">
    Pull the latest release.

    <CodeGroup tabs="Docker,CLI">
    ````bash
    docker pull wesleyn/docs:latest
    docker run -p 3000:3000 wesleyn/docs:latest
    ````

    ````bash
    wesleyn update --version 1.3.0
    ````
    </CodeGroup>
  </Step>
  <Step title="Verify" icon="check-circle">
    Test key features like editing and search.
  </Step>
  <Step title="Rollback if Needed" icon="rotate-ccw">
    Restore from backup if issues arise.
  </Step>
</Steps>
